1.VI编辑器：

Visual Interface：可视化接口，类似于windows中的记事本，但是强大了太多。
VI -> VIM：升级版本，兼容所有VI的命令。

VIM支持多级撤销
VIM跨平台
VIM支持语法高亮
VIM支持图形界面

2.VI编辑器的操作模式：

Command Mode：命令模式
Insert Mode：输入模式
Last Line Mode：底行模式(尾行,末行)

命令模式和底行模式比较类似，有时也归到同一类中。

VIM编辑器的命令格式：

vim 文件名    打开文件(如果文件不存在则自动创建)    如：vim 123.txt -> i(切换到输入模式) -> esc(切换到底行模式) -> :wq(保存并退出)
vim + 文件名    打开文件，并把光标移到最后一行
vim +3 文件名    打开文件，并把光标移到第3行，如果填写行数超过总行数，那么光标移到最后一行
vim +/xxx 文件名    打开文件，把光标移到xxx第一次出现的位置
vim aa bb cc    一次性打开多个文件，如果文件不存在则创建该文件，在底行模式中输入 :n 切换各个文件

命令和底行模式常用指令：

底行：
:w    将文件的修改从内存写入硬盘
:q    退出当前vim编辑器
:!    强制执行
:ls   列出当前编辑器中打开的所有文件
:n/:N    切换到后一个文件/切换到前一个文件
:15   将光标快速定位到第15行
/xxx  从光标所在位置开始搜索，将光标移到xxx第一次出现的位置
/?xxx    从光标所在位置开始向前搜索，将光标移到xxx第一次出现的位置  

可以配合使用

命令模式：

h    光标左移
j    光标下移
k    光标上移
l    光标右移

ctrl + f    向下翻页
ctrl + b    向上翻页
ctrl + d    向下翻半页
ctrl + u    向上翻半页

dd    删除光标所在行
o     在光标所在行的下方插入一行并切换到输入模式
yy    复制光标所在的行
p     在光标所在行的下方粘贴
P     在光标所在行的上方粘贴
